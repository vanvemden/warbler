What is Warbler?

Warbler is a Twitter clone, named after a colorful bird. I started this project during Rithm School Full-Stack Web Developer boot camp, which I graduated from August 2020.

The code is available on my GitHub project page. The app is made with Python, HTML, CSS, Flask, Bcrypt, SQLAlchemy, WTForms, Jinja, and a few lines of JavaScript.
